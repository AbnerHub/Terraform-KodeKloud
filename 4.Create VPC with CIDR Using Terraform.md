## 📌 Create VPC with CIDR Using Terraform

Create a VPC named `xfusion-vpc` in `us-east-1` region with `192.168.0.0/24` IPv4 CIDR using terraform.


The Terraform working directory is `/home/bob/terraform`. 
Create the `main.tf` file (do not create a different `.tf` file) to accomplish this task.


## 🚀  Solution Steps


```
resource "" "xfusion-vpc"{
    cidr_block = "192.168.0.0/24"
    tags = {
        Name = "xfusion-vpc"
    }

}
```
