```markdown
## 📌 Task
For this task, create an EC2 instance using Terraform with the following requirements:

1. The EC2 instance must use the value `nautilus-ec2` as its Name tag, which defines the instance name in AWS.

2. Use the `Amazon Linux` `ami-0c101f26f147fa7fd` to launch this instance.

3. The Instance type must be `t2.micro`.

4. Create a new RSA key named `nautilus-kp`.

5. Attach the default (available by default) security group.

The Terraform working directory is `/home/bob/terraform`. Create the `main.tf` file (do not create a different `.tf` file) to provision the instance.


## 🚀 Steps Solution

```
# Generate locally a private RSA key
resource "tls_private_key" "nautilus-kp"{
    algorithm = "RSA"
    rsa_bits = "4096"
}

# Create an AWS public key 
resource "aws_key_pair" "nautilus-kp"{
    key_name = "nautilus-kp"
    public_key = tls_private_key.nautilus-kp.public_key_openssh
}

# Save the pirvate key locally 
resource "local_sensitive_file" "private_key"{
    filename = "/home/bob/nautilus-kp.pem"
    content = tls_private_key.nautilus-kp.private_key_pem
    file_permission = "0600"
}

# Get default SG using data source
data "aws_security_group" "default"{
    name = "default"
}

# Create an ec2 instance
resource "aws_instance" "nautilus-ec2" {
    ami = "ami-0c101f26f147fa7fd"
    instance_type = "t2.micro"
    #attach the created aws public key created
    key_name = aws_key_pair.nautilus-kp.key_name
    # Attach the default vpc
    vpc_security_group_ids = [data.aws_security_group.default.id]
    
    tags = {
        Name = "nautilus-ec2"
    }
}
